# VN Map - Technology Stack

## Frontend Framework
- **Next.js 14.1.3**: React framework with pages router
- **React 18.2.0**: Core UI library  
- **TypeScript 4.9.5**: Type safety with strict configuration

## Mapping & Visualization
- **Leaflet 1.9.3**: Open-source mapping library
- **React Leaflet 4.2.1**: React wrapper for Leaflet
- **Leaflet MarkerCluster 1.5.3**: Marker clustering functionality

## UI Components & Styling
- **Tailwind CSS 3.4.17**: Utility-first CSS framework
- **Radix UI**: Comprehensive accessible component library
- **Lucide React 0.532.0**: Icon library
- **Class Variance Authority**: Component variant management
- **Tailwind Merge**: Conditional styling utility
- **Tailwindcss Animate**: Animation utilities

## State Management & Data
- **React Hook Form 7.61.1**: Form handling
- **React Context**: For map state management
- **Custom hooks**: useMapContext, useMarkerData, useLeafletWindow

## Development Tools
- **ESLint**: Code linting with Airbnb + TypeScript config
- **Prettier**: Code formatting
- **PostCSS**: CSS processing
- **Autoprefixer**: CSS vendor prefixes

## Build & Development
- **pnpm**: Package manager (has pnpm-lock.yaml)
- **Node.js**: Runtime environment
